# etecoons

Clean, secure, and blazing-fast self-hosted utilities built in Rust.

## Projects

| Project | Description |
| :--- | :--- |
| **[beam](https://github.com/etecoons/beam)** | High-speed, secure file dropper transfer utility. |
| **[grid](https://github.com/etecoons/grid)** | Self-hosted Kanban board application. |
| **[pad](https://github.com/etecoons/pad)** | Collaborative real-time scratchpad. |
| **[todo](https://github.com/etecoons/todo)** | Minimalist task management tool. |
| **[trace](https://github.com/etecoons/trace)** | Network diagnostic, WHOIS, IP, and ASN lookup utility. |
| **[pulse](https://github.com/etecoons/pulse)** | Real-time system monitoring panel. |
| **[scan](https://github.com/etecoons/scan)** | Planetary hazard sector scanner (Minesweeper clone). |

## Deployment

Deploy any of these applications using **Docker / Podman** containers, or install them on **Unraid** using the official templates repository:

* **Unraid Templates:** [etecoons/unraid-apps](https://github.com/etecoons/unraid-apps)
* **Docker Registry:** Pull namespace `etecoons/` or `ubermetroid/`
