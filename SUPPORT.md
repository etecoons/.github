# Support Guide

Welcome to the etecoons community! If you need help with any of our tools (such as `trance` or `morphball`), please refer to the options below.

## 🔍 Search Existing Issues and Documentation

Before opening a new issue or asking a question, please:
- Search the issue tracker of the specific project repository (e.g., [etecoons/trance](https://github.com/etecoons/trance/issues) or [etecoons/morphball](https://github.com/etecoons/morphball/issues)).
- Review the `README.md` and any documentation under the repository's `docs/` folder.

## 🐛 Report a Bug

If you've found a bug, please open a bug report in the repository where the issue occurred:
- [Report a bug for trance](https://github.com/etecoons/trance/issues/new)
- [Report a bug for morphball](https://github.com/etecoons/morphball/issues/new)

## 💡 Suggest a Feature

Have an idea for a new feature or enhancement? We'd love to hear it! Open a feature request in the relevant repository.

## 🔒 Security Vulnerabilities

If you discover a security vulnerability, please do **not** open a public issue. Instead, refer to our [SECURITY.md](SECURITY.md) to report it responsibly.
